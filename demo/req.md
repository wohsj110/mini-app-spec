# Demo requirement: Export Watermark Toggle

> Fixture requirement for the public demo spec. No real product, no real business.

1. The export settings screen gains a "watermark" toggle, default ON.
2. When the toggle is OFF, exported images contain no watermark.
3. A watermark opacity slider (0–100%) adjusts the watermark in real time.
4. The toggle state persists across sessions (kill & relaunch keeps the last choice).
